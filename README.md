# Tradutor de Linguagem por Voz e Texto

## Visão Geral do Projeto

Este projeto implementa um sistema de tradução automática que permite a conversão de texto e voz entre diferentes idiomas, utilizando tecnologias de processamento de linguagem natural.


### Backend de Tradução

Para a etapa de tradução automática, foi utilizada a biblioteca open-source **Deep Translator**, adotada em virtude da indisponibilidade de créditos na API da OpenAI. Trata-se de uma ferramenta leve que permite a tradução de textos entre idiomas por meio de serviços públicos de tradução, garantindo compatibilidade com o ambiente e a manutenção do fluxo lógico proposto.
